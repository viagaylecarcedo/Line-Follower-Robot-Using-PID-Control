# Line-Follower-Robot-Using-PID-Control
## 📌 Project Overview
This project is an **Arduino-based Line Follower Robot** that uses a **PID (Proportional–Integral–Derivative) control algorithm** to follow a line with high accuracy and stability.

Unlike basic line followers, this robot continuously adjusts motor speeds using real-time sensor feedback, allowing smoother movement and better performance on curves and sharp turns.

---

## 🎯 Objectives
- Design an autonomous line-following robot  
- Implement a **closed-loop control system**  
- Apply **PID control algorithm** for precision  
- Improve tracking accuracy and stability
